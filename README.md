# DuelDansLeCiel
Second projet tutoré en L1 Informatique

La proposition de projet provient de Joseph Razik (http://razik.univ-tln.fr/duel-dans-le-ciel.html)
Le projet n'est pas totalement codé pour l'instant.
Il a été codé et organisé pour être le plus modifiable possible en étant le plus propre possible en terme structure.
Tout ce qui touche à la fenêtre graphique provient d'une librairie nommée cng développé par Christian Nguyen et Joseph Razik.
Cette librairie a toute fois été modifié pour répondre plus amplement à mes besoins.

/!\ THOSE INSTALLATIONS ARE REQUIRED /!\

  - Python 3.9.x
  - install Pillow module (python -m pip pyglet)
  - install pyglet module (python -m pip Pillow)
